Блок навигации

  • Ширина 15 em
  • Должен присутствовать цвет фона
  • Распологается слева
  • Блок навигации наполните контентом достаточным, чтобы по высоте он начинал захватывать вторую статью, как показано в данном примере

Статья 1

Размер блока: Занимает всю доступную ширину Должен присутствовать цвет фона Во всех статьях сделайте отступ от соседней .2em, при помощи свойства margin-bottom

Статья 2

Размер блока: Занимает всю доступную ширину Должен присутствовать цвет фона Во всех статьях сделайте отступ от соседней .2em, при помощи свойства margin-bottom Картинка вставлена при помощи тега, плавание органищовано по правому краювнутриданной статьи для этой картинки.

Нижний блок, подвал. Занимает всю доступную ширину.Высота определяется помещенным внутрь контентом.
